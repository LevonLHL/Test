## 公式

### 等价无穷小

- x-sin x ~ (1/6)x³
- x-arcsin x ~ -(1/6)x³
- x-arctan x ~ (1/3)x³
- x-tan x ~ -(1/3)x³
- x-㏑(1+x) ~ (1/2)x²
- (a^x)-1 ~ x㏑a
- ((1+x)^a)-1 ~ ax
- (√(1+x)) - (√(1-x)) ~ x

### 导数

- (cot x)'=-csc² x
- (sec x)'=(sec x)tan x
- (csc x)'=-(csc x)cot x
- (arccos x)'=-1/√（1-x²）
- (arccot x)'=-1/(1+x²)

### 积分

- ∫ tan x dx=-㏑|cos x| +C
- ∫ cot x dx=㏑|sin x| +C
- ∫ sec x dx=㏑|(sec x)+tan x| +C
- ∫ csc x dx=㏑|(csc x)-cot x| +C
- ∫ csc² x dx=-cot x +C
- ∫ 1/(a²+x²) dx=(1/a)arctan (x/a) +C
- ∫ 1/(a²-x²) dx=(1/2a)㏑|(a+x)/(a-x)| +C
- ∫ 1/√(a²-x²) dx=arcsin (x/a) +C
- ∫ 1/√(a²±x²) dx=㏑|x+√(a²±x²)| +C

### 三角函数

- cos² x=(1+cos 2x)/2

- sin² x=(1-cos 2x)/2

- sin 2x=2(sin x cos x)

- cos 2x=cos² x-sin² x=2(cos² x)-1

- tan 2x=(2tan x)/1-tan² x

- sin x=cos x=cos (π/2-x)

- arcsin (1/x)=arccos x

- arccos (1/x)=arcsin x

- arcsin x+arccos x=arctan x+arccot x=π/2
