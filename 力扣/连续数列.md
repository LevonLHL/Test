给定一个整数数组，找出总和最大的连续数列，并返回总和。

**示例：**

```
输入： [-2,1,-3,4,-1,2,1,-5,4]
输出： 6
解释： 连续子数组 [4,-1,2,1] 的和最大，为 6。
```



---

~~~c
int maxSubArray(int* nums, int numsSize){
    int max=nums[0];
    int temp=0;
    for(int i=0;i<numsSize;i++)
    {
        temp+=nums[i];
        if(temp>max)    max=temp;
        if(temp<0)  temp=0;
    }
    return max;
}
~~~
